## 特性

- 支持使用 Fluent API 与 JSON。
- 基于 EmbeddedView，没有 HostElement，支持嵌套布局。
- 按需导入，可摇树。
- 建立在 Angular 响应式表单之上。
- 与 [Ant Design](https://ng.ant.design) 集成。
