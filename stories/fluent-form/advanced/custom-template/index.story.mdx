# Custom Template

使用 `FluentTemplate` 指令在 `FluentForm` 组件下注册自定义模板。

使用 `headless` 或 `template` 图示来渲染自定义模板。
两者不同的是，`headless` 是一个**表单控件**图示，除了可以显示模板，它还将形成表单模型；而 `template` 图示仅可显示模板，不会存在于表单模型中。
