## 特性

- 支持使用 Fluent API 与 JSON。
- 类型安全的表单配置。
- 建立在 Angular 响应式表单之上。
- 基于 `ng-zorro-antd` 的表单控件与栅格布局。

## 先决条件

- [Angular](https://angular.io) >= 13.0.0
- [ng-zorro-antd](https://ng.ant.design) >= 13.0.0
